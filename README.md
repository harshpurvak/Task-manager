# Task Manager

This is my small JavaScript project for practice.

In this project I made a simple task manager where we can add tasks with title,
description and category. We can also complete, edit, delete and search tasks.
I also added dark mode.

I used this project to practice:

-HTML
-CSS
-JavaScript
-DOM manipulation
-events
-event bubbling and capturing

## How to run

Just open `index.html` in the browser



This is not a big professional project, just a learning project.
